Challenge:An Executable file present in the home directory which we can be used to retrieve the password for the next level,In order to retrieve the password you need to by pass the permission restrictions by creating a link.

Commands_Used: ln ( Symbolic link)

#Explanation:A symbolic link (or symlink) is a type of file that acts as a pointer to another file or directory. It allows you to create a reference to a file or directory in a different location without duplicating the actual content.

6.Create temporary directory using the "mktemp -d" and change your directory to that temp directory

7.Create file using the "touch" command and give the space in the name like this "password file.txt"

#Now if we run the exe file once again (./printfile /tmp/tempdirectoryname/password file.txt) we observer that it gives the output as "file.txt" not found, in other words it didn't ask for the full file name it ignored the password and from this we can create a "symbolic link"

8.To create symbolic link (Clear the terminal first using the {clear} command) and then type the following command

"ln -s /etc/leviathan_pass/leviathan3 /tmp/tempdirectoryname/password"

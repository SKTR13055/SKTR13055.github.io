myname=$(whoami)
mytarget=$(echo I am user $myname | md5sum | cut -d ' ' -f 1)

echo "Copying passwordfile /etc/bandit_pass/$myname to /tmp/$mytarget"

cat /etc/bandit_pass/$myname > /tmp/$mytarget

-> In this shell script it shows the inner working of the script 

here $(whoami) is basically telling to the run the command "whoami" and store that value in "myname" variable.

->Next is running the echo command and using the "myname" variable {which is basically bandit22}
and after that it creates an "md5" hash of the text.

->Later on it copies the pasword from the usernames password directory to the temp file (which is the md5 hash of the username)

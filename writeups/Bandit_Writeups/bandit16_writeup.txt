How to Solve OvertheWire Bandit Level 15 -> 16 level?

Objective:The password for the next level can be retrieved by submitting the password of the current level to port 30001 on localhost using SSL/TLS encryption.

Helpful note: Getting “DONE”, “RENEGOTIATING” or “KEYUPDATE”? Read the “CONNECTED COMMANDS” section in the manpage.

Commands needed to solve: ssh, telnet(possible), nc(possible), socat, openssl(used), s_client(used), nmap(used), netstat, ss(not used)

1.in the Commands needed section I mentioned two keywords used,possible 
  used: used in this level to solve the problem
  possible: It is possible to use this command to solve the problem

2.To solve this level we first need to know what does every command does (skip to 3 if you know what every command does) we can use "whatis" or "man" command to know
    
  ssh(secure shell): - OpenSSH remote login client
  
  telnet: - user interface to the TELNET protocol #Not used in this level

  nc/ncat/netcat: - TCP/IP swiss army knife #Not/Possible way to use in this level

  socat:- Multipurpose relay (SOcket CAT) #Not Used in this level

  s_client (1ssl) - OpenSSL application commands {This command implements a generic SSL/TLS "client which connects to a remote host using SSL/TLS".
       It is a very useful diagnostic tool for SSL servers}
  
  nmap:- Network exploration tool and security / port scanner

 ss (8)               - another utility to investigate sockets
  
 netstat (8)          -  Print  network connections, routing tables, interface statistics, masquerade connections, and multicast memberships

 openssl{open SecureSocketLayer} (1ssl)       - OpenSSL command line program {type man openssl for more details}

-> Basically in this challenge you are given a file(data.txt) and you have to use a hex editor to understand and decompress it one by one for example: "Like a Chocolate wrapped under several Wrappers and you are unwrapping it one by one to get the delicious chocolate :p"

->Now Most of the commands used are tar,gzip,bzip and xxd or hexeditor
*tar is basically an archiving utility in other words you can say a file is archived under it.
*gzip and bzip2 are like file compressors and decompressors but you need both of them.
*xxd is a hexdumper this basically shows unreadable strings into hexadecimal numbers similarly for hexdump(my favorite hex editor)

Solution:As I said before the solution will be divided into 3 phases 

1st Phase (Moving the file into temporary)

1.Login to the level using the password obtained from the previous level.
2.Now the Level instruct us to make a temporary folder using the "mktmp -d" command the output will be displayed as the temp file path.
3.Copy the file using the "cp" command and paste it into the temporary folder, rename the file using the "mv" command like this mv "<old_filename> <new_filename>" ( Ignore the double quotes)

#End of the 1st Phase.

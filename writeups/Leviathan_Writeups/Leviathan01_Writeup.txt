#Explanation:There is a particular file called the 'check' which is an executable file and from that executable you have to get the password for 'leviathan2', Analyzing how the program works and finding out how it works is called "Reverse Engineering". This really important in terms of malware analysis in order to find out how the malware works, I personally this is a cool thing to learn and keep it in your arsenal.

-> "ltrace" is a debugging utility in Linux that traces calls made by a program to shared libraries. It works by intercepting and recording dynamic library calls, allowing users to analyze how a program interacts with its libraries during execution.

$key = $_REQUEST['needle'] ?? '';
if (preg_match('/^[a-zA-Z]+$/', $key)) { // Whitelist validation
    $lines = file('dictionary.txt');
    $results = preg_grep("/$key/i", $lines);
    foreach ($results as $line) {
        echo htmlspecialchars($line) . "<br>"; // Sanitize output
    }
} else {
    echo "Invalid input.";
}

This solution validates the input against a whitelist of alphabetic characters, uses PHP functions to read and search the file, and sanitizes the output to prevent XSS.

#Explanation

When you open a website, your computer (the client) asks the website's server for the page. This is called a request, and the server sends back the response — usually the website’s content (like HTML).

This back-and-forth uses a system called HTTP.
What does a request include?

An HTTP request usually has:

    Method: what you want to do (like GET to load a page).

    URL: the web address you want.

    Headers: extra info like:

        User-Agent: your browser info

        Accept: what kind of files you can receive (HTML, images, etc.)

        Authorization: login info (if needed)

        Referer: the page you came from

Your browser does all this in the background, but you can also send requests yourself.

4.Type the command: find ./maybehere* (or) maybehere*/  -type f -exec ls -la {} + | grep 1033

# now the explanation of each command goes like this the .(dot) represents the current working directory by adding the ./ you are indicating that you want to look for items in the current directory
->maybehere* (asterisk) is a wildcard that matches any characters following the 'maybehere' This means it will match any file or directory in the current directory that starts with maybehere

->The -type f stands for the data type where f stands for "file" 
-> The -exec is telling to execute the next command which is ls -la {} + (list out the files along with the hidden files and their permissions) the flower brackets is a placeholder for the files during the execution and the "+" check and more files present in the directories 
->Finally the | (aka Piping) this is used to transfer the output of the left command to the right command grep command

-> The "grep" is a basic command which captures characters given by the user in this case it is 1033(bytes given as a clue in the challenge)

Title: Bandit Level 14 → Level 15

Challenge: The password for the next level can be retrieved by submitting the password of the current level to port 30000 on localhost.

Commands Used in this level: nc or netcat

#Explanation: nc aka netcat is a command which allows to "read and write" data over a network connection. Can be used for TCP(Transmission Control Protocol) and UDP (User Datagram Protocol).

To connect to a service using the nc command the following syntax is to be followed nc <host:port>. You can also create a server which can listen to incoming packets like this nc -l <port> [Where l basically stands for listen]
